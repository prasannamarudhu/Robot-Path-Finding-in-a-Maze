◆ printFinalScenario()

void Scenario::printFinalScenario ( bool  goalReached)

This function prints final maze.

Once all the navigation is completed, irrespective of success/failure, this function is called to print the final result. This prints map with the same format as input characeters along with traversed path with a '+' character.
(#) -> obstacle,
( ) -> unexplored free-space,
(+) -> explored location,
(S) -> Start node,
(G) -> Goal node.
If goal is reached, only the path leading to goal is printed, else partial path is printed.

Parameters
goalReachedvalue indicating if the goal is reached or not.
Returns
Void.
